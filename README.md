# 🇸🇪 Sweden Weather Analysis

EDA project analyzing weather patterns across 5 Swedish cities using real-time data.

## Cities Analyzed
- Göteborg, Stockholm, Malmö, Uppsala, Linköping

## What's Inside
- Real weather data fetched from Open-Meteo API (last 3 months)
- Temperature, precipitation, wind speed analysis
- City comparisons and trend visualizations

## Tools Used
- Python, Pandas, Matplotlib, Seaborn, Requests

## Key Findings
- Coldest day: Linköping -17.9°C (Feb 15, 2026)
- Warmest day: Malmö +13.5°C (Mar 8, 2026)
- Windiest day: Göteborg 46.8 km/h (Mar 13, 2026)
